NVM

2019-08-22 1 min read # Node.js

Node Version Manager - POSIX-compliant bash script to manage multiple active node.js versions

1. Install

$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash

$ nvm install node
$ node -v

2. List

$ nvm list
$ nvm list available

3. Use

$ nvm use 12.9.0

4. Uninstall

$ nvm uninstall

5. Removal

$ rm -rf "$NVM_DIR"
