Mac-Homebrew

2019-08-23 Mac

The missing package manager for macOS (or Linux)

1. Install

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

2. Usage

$ brew install wget
$ brew cask install firefox

$ brew uninstall wget
$ brew cask uninstall firefox

3. Search

$ brew search git

4. Info

$ brew info git

5. List

$ brew list
