Rails-Account-Migrate

2019-09-12 Rails

Data Migration is the process of migrating data from one data environment to another.

tips： 表-命名习惯小写复数，模型-习惯命名大写单数

1. 表结构

表名称：accounts

字段名字 字段说明 字段类型 备注
name 用户名称 string 必填
email 注册邮箱 string 必填
password 密码 string 必填
role 用户角色（控制权限） integer 0普通用户 1管理员 2超级管理员
status 用户状态（是否激活） integer 0已激活 1未激活

2. 迁移文件
