Mac-Vagrant-Ubuntu

2019-08-20 Mac

Development Environments Made Easy

1. Install

$ brew cask install virtualbox
$ brew cask install vagrant
$ brew cask install vagrant-manager

2. Init

$ mkdir rails-va
$ cd rails-va
$ vagrant init ubuntu/bionic64

3. Usage

$ cd rails-va
$ vagrant status
$ vagrant up
$ vagrant ssh
$ vagrant reload
$ vagrant halt
